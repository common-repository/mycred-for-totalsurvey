myCred for Total Survey is a WordPress plugin for gamified surveys. Complete the survey to get rewards in the form of points, ranks, or badges.

== Description ==

[TotalSurvey](https://wordpress.org/plugins/totalsurvey/) is a WordPress plugin that let you create and integrate Surveys easily, TotalSurvey is made to be easy and powerful, Although it's very simple to use, It provides big number of options and features to have full control over the Surveys. This plugin allows you to reward users with points for Completing Surveys.

= myCred for TotalSurvey Features =
Here are the most notable features of myCred for TotalSurvey.

* Award points for Completing Surveys.
* Badge support.
* Translation ready.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for TotalSurvey
3. Find myCred for TotalSurvey and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
